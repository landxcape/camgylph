# Scope Plan: v1.5.0 Distribution Automation

- Status: Planned
- Date: 2026-03-02
- Target milestone tag: `v1.5.0`

## Objective

Automate GitHub release and Homebrew tap updates to reduce manual publish steps and release drift.

## Planned Steps

1. Add GitHub Actions workflow for tag-triggered release validation.
2. Add workflow to compute Homebrew tarball checksum for new tags.
3. Open/update tap formula PR automatically (or push directly to tap main).
4. Add release checklist docs for manual fallback when CI is unavailable.

## Acceptance Criteria

- New version tag triggers repeatable release validation.
- Homebrew formula update path is automated and deterministic.
- Documentation includes rollback/fallback process.
