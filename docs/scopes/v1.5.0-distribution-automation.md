# Scope: v1.5.0 Distribution Automation

- Status: Complete
- Date: 2026-03-02
- Milestone tag: `v1.5.0`

## Goal

Automate release validation and Homebrew tap sync on version tags, with a clear manual fallback process.

## Completed

- Added tag-triggered release validation workflow:
  - `.github/workflows/release-validate.yml`
  - Validates tag/Cargo version match, formatting, tests, and release build.
- Added tag-triggered Homebrew tap sync workflow:
  - `.github/workflows/sync-homebrew-tap.yml`
  - Computes source tarball checksum and updates tap formula.
  - Skips safely when required repository secret is not configured.
- Added manual release fallback checklist:
  - `docs/release-checklist.md`

## Why This Scope

- Reduces manual release drift between GitHub and Homebrew.
- Makes release safety checks repeatable.
- Keeps a documented fallback path for reliability.

## Why Not More in This Scope

- No bottle automation added yet (separate delivery concern).
- No artifact packaging workflow yet (focus was validation + tap sync).

## Validation

```bash
cargo fmt
cargo check
cargo test
```

Expected:

- New tags trigger validation workflow.
- If required secret exists, tap formula auto-updates.
- If required secret is missing, tap sync skips with clear message.
